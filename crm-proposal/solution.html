Proposed Solution

Four integrated modules addressing all 24 operational gaps.

System Design

Four Core Modules

M1 — Data Pipeline
  • Upload carrier Excel/CSV in any format
  • Auto-detect column mapping per carrier
  • Save mapping templates for reuse
  • Normalize product names & phone numbers
  • Match/merge duplicate customers across carriers
  • Flag premium changes (old vs new)
  • Compute days-to-expiry & grace period
M2 — Agent CRM
  • Auto-assign policies by round-robin / rules
  • Click-to-call with structured disposition form
  • 4 mandatory fields per call log (see taxonomy below)
  • Callback scheduling with calendar + reminders
  • Full call history timeline per customer
  • Policy card: premium, product, SI, expiry, carrier
  • Daily task queue with priority sorting
M3 — Analytics Dashboard
  • Real-time renewal rate by carrier, product, agent
  • Pipeline funnel: Pending → In Progress → Renewed / Lost
  • Premium volume & commission estimates
  • Agent leaderboard & daily call targets
  • Aging analysis: stale leads flagged
  • Loss reason breakdown (top 5 reasons)
  • Exportable MIS reports (CSV)
M4 — Communication
  • Pre-call SMS/WhatsApp reminder (bulk or single)
  • Renewal due notification to customer
  • Expiry alert escalation to manager
  • Template library for common messages
  • Delivery status tracking
