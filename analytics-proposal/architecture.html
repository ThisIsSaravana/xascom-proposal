  • Designed for insurance broker data workflow
  • Carrier file import with saved templates
  • Insurance-specific KPIs and charts
  • No per-user licensing fees
  • Full data ownership (your database)
  • Unlimited users, unlimited records

  • Generic dashboards — no broker workflow

Architecture · Slide 4
Security & Compliance

Data Protection Model

Row-Level Security

Supabase RLS ensures data access is controlled at the database layer. Admin sees all, viewers see assigned data.

Authentication

Supabase Auth with email + password. JWT-based sessions. Secure login flow.

Role-Based Access

Two roles: Admin (full access, manage users) and Viewer (read-only dashboard access).

HTTPS / SSL

All traffic encrypted via Vercel SSL. No additional certificate management needed.

Automated Backups

Supabase daily database backups. Point-in-time recovery available.

Upload Audit Trail

Every file upload logged with user, timestamp, carrier, and record count.
