Proposed Solution

Four modules addressing all 12 data and analytics gaps.

System Design

Four Core Modules

M1 — Data Ingestion
  • Drag-drop file upload (Excel / CSV)
  • Visual column mapping UI
  • Save mapping template per carrier (map once, reuse forever)
  • Manual entry form for individual policies
  • Data validation — missing fields, duplicates, format errors
  • Preview mapped data before confirming import
  • Full upload history with date, carrier, record count
M2 — Data Management
  • Single searchable table replacing all 192 sheet tabs
  • Filter by carrier, product, status, date range, premium band
  • Quick search by policy number, customer name, or mobile
  • Inline editing — click any field to update
  • Bulk actions — select multiple, change status, export
  • Show/hide columns per preference
  • Export filtered data as CSV / Excel
M3 — Analytics Dashboard
  • 6 KPI cards — total policies, premium, carriers, renewal rate, customers, avg premium
  • Carrier breakdown chart (policies & premium per carrier)
  • Product mix donut (COMP, FHO, PA, Arogya, etc.)
  • Premium trend line (monthly growth/decline)
  • Renewal rate by month (bar chart)
  • Fresh vs Renewal split & premium band distribution
  • Top 10 customers by premium & carrier growth YoY
  • Global filters — date, carrier, product, status
M4 — Reports & Export
  • Monthly MIS report — summary of policies, premium, renewals
  • Carrier performance report — per carrier ranking
  • Product mix report — distribution with breakdowns
  • Renewal pipeline — expirations in next 30/60/90 days
  • Customer report — policy count, total premium, carriers
  • Export to CSV, Excel, or PDF
